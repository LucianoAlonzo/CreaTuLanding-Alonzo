# Urucel — Entrega 2: 

## Resumen
Esta entrega implementa la navegación de la tienda. Se añadieron rutas para el catálogo general, catálogo filtrado por categoría y la vista de detalle de cada producto. Los datos se simulan con promises para simular llamados asíncronos.

## Rutas implementadas
- `/` — Catálogo completo 
- `/category/:categoryId` — Catálogo filtrado por categoría
- `/item/:id` — Vista de detalle de un producto 
- `*` — Página 404 

## Dependencias
react-router-dom react-icons bootstrap
